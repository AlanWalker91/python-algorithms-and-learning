# LeetCode 热题 100 - 算法教练式详解

> 本文档按照 LeetCode 官方「热题 100」学习计划整理，共 17 个分类、100 道题目。
> 每道题均按照「教练引导式」方法讲解，不直接给答案，而是引导你从思考到解题的完整过程。

## 每道题的讲解结构

1. **核心考点** - 这道题考什么？属于哪一类？
2. **解题思路** - 用最直观的方式讲清楚
3. **暴力→优化** - 从最笨的方法推导到最优解
4. **关键思维点** - 每一步为什么这样做？
5. **代码实现** - Python 代码 + 逐行解释
6. **套路总结** - 下次怎么一眼识别这类题

## 刷题建议

1. **按分类刷** - 同一类题目放在一起练，建立模式识别能力
2. **先想再看** - 每道题至少独立思考 15 分钟再看解析
3. **暴力优先** - 先写出暴力解，再思考如何优化
4. **总结套路** - 每做完一类题，回顾共性，形成自己的模板
5. **间隔复习** - 隔 3 天、7 天、14 天分别复习一次
