# 滑动窗口

滑动窗口是双指针的一种特殊形式，核心思想是维护一个窗口（连续子数组/子串），通过左右指针的移动来高效地遍历所有可能的子区间。

---

## 题目1：3. 无重复字符的最长子串

### 1. 核心考点

**滑动窗口 + 哈希集合**。本题考查的是如何用滑动窗口维护一个「无重复字符」的区间，并在遍历过程中动态更新最大长度。属于经典的「可变长度滑动窗口」问题。

### 2. 解题思路（给新手讲）

想象你在读一本书，你想找到最长的一段文字，其中每个字母都不重复。

你可以用两根手指夹住一段文字：
- **右手指**不断往右移动，把新字符纳入范围
- 每当发现新字符已经在范围内出现过了，就把**左手指**往右挪，直到把那个重复的字符排除出去

在整个过程中，你随时记录两根手指之间最远的距离，那就是答案。

关键点：右指针只会往右走，左指针也只会往右走，两个指针都不会回头。所以虽然有两个循环（外层右指针、内层左指针），但总的移动次数是 O(n) 级别的。

### 3. 从暴力到优化的推导

#### 暴力解法

枚举所有子串，对每个子串检查是否有重复字符：

```python
def lengthOfLongestSubstring(s: str) -> int:
    n = len(s)
    ans = 0
    for i in range(n):           # 枚举起点
        for j in range(i, n):    # 枚举终点
            sub = s[i:j+1]
            if len(sub) == len(set(sub)):  # 无重复
                ans = max(ans, j - i + 1)
            else:
                break            # 已重复，更长的也会重复
    return ans
```

时间复杂度：O(n^2)，空间复杂度：O(n)。

#### 优化思路

暴力解法中，当我们发现 `s[i..j]` 有重复时，直接 break 去尝试 `i+1` 作为新起点。但其实：
- 如果 `s[i..j]` 无重复而 `s[j+1]` 导致重复，那么新起点不需要从 `i+1` 重新扫描到 `j+1`
- 我们只需要把左边界往右移，直到窗口内没有重复字符

这就是滑动窗口的核心思路：**右指针扩展窗口，左指针收缩窗口，两个指针都只向右移动**。

#### 最优解法

使用一个哈希集合（set）记录窗口内的字符，右指针向右扩展时：
- 如果新字符不在集合中，加入集合，更新最大长度
- 如果新字符在集合中，不断移动左指针并从集合中移除字符，直到窗口内无重复

时间复杂度：O(n)，空间复杂度：O(min(n, 字符集大小))。

### 4. 关键思维点

1. **窗口的含义**：`s[left..right]` 始终保持无重复字符
2. **为什么不会漏解**：右指针遍历了每个位置作为窗口右端点，左指针保证了窗口的合法性
3. **为什么是 O(n)**：左指针和右指针各自最多移动 n 次，总操作 2n 次
4. **进一步优化**：可以用哈希表记录字符最后出现的位置，左指针可以直接跳到重复字符的下一个位置，避免逐步移动

### 5. 代码实现（Python）

```python
def lengthOfLongestSubstring(s: str) -> int:
    # 哈希集合，记录当前窗口内的字符
    char_set = set()
    # 左指针
    left = 0
    # 记录最长长度
    ans = 0

    # 右指针遍历整个字符串
    for right in range(len(s)):
        # 如果当前字符已经在窗口中，收缩左边界
        while s[right] in char_set:
            char_set.remove(s[left])  # 移除左边界字符
            left += 1                 # 左指针右移

        # 此时窗口内无重复，将当前字符加入集合
        char_set.add(s[right])
        # 更新最大长度
        ans = max(ans, right - left + 1)

    return ans
```

**进阶版本**（哈希表记录位置，左指针直接跳转）：

```python
def lengthOfLongestSubstring(s: str) -> int:
    # 哈希表：字符 -> 该字符最后出现的索引
    last_seen = {}
    left = 0
    ans = 0

    for right in range(len(s)):
        ch = s[right]
        # 如果字符之前出现过，并且在当前窗口内
        if ch in last_seen and last_seen[ch] >= left:
            # 直接跳到重复字符的下一个位置
            left = last_seen[ch] + 1

        # 更新字符的最后出现位置
        last_seen[ch] = right
        # 更新最大长度
        ans = max(ans, right - left + 1)

    return ans
```

### 6. 套路总结

**识别信号**：题目要求「最长/最短的满足某条件的连续子串/子数组」。

**滑动窗口模板**：

```python
def sliding_window(s):
    left = 0
    window = {}  # 或 set，维护窗口状态
    ans = 0

    for right in range(len(s)):
        # 1. 扩展：将 s[right] 加入窗口

        # 2. 收缩：当窗口不满足条件时，移动左指针
        while 窗口不满足条件:
            # 将 s[left] 移出窗口
            left += 1

        # 3. 更新答案（在窗口合法时更新）
        ans = max(ans, right - left + 1)

    return ans
```

**核心要点**：搞清楚三件事 -- (1) 窗口何时扩展，(2) 窗口何时收缩，(3) 答案何时更新。

## 题目2：438. 找到字符串中所有字母异位词

### 1. 核心考点

**固定长度滑动窗口 + 字符频次比较**。本题考查的是如何用定长窗口在字符串中滑动，判断窗口内的字符组成是否与目标字符串完全一致。属于「定长滑动窗口」问题。

### 2. 解题思路（给新手讲）

「字母异位词」就是字母完全相同，但顺序可以不同的词。比如 "abc" 的异位词有 "abc"、"acb"、"bac"、"bca"、"cab"、"cba"。

想象你有一把特殊的尺子，长度恰好等于目标词 p 的长度。你拿着这把尺子在字符串 s 上从左往右滑：
- 每滑到一个位置，你检查尺子盖住的那段文字，里面每个字母的个数是不是和 p 一模一样
- 如果一样，就记下这个起始位置

关键优化：你不需要每次都重新统计整段文字的字母个数。每次尺子往右滑一格，左边掉出去一个字母、右边新进来一个字母，你只需要更新这两个字母的计数就行了。

### 3. 从暴力到优化的推导

#### 暴力解法

对 s 中每个长度为 `len(p)` 的子串，统计字符频次并与 p 的频次比较：

```python
from collections import Counter

def findAnagrams(s: str, p: str) -> list:
    ans = []
    p_len = len(p)
    p_count = Counter(p)

    for i in range(len(s) - p_len + 1):
        # 每次都重新统计子串的字符频次
        if Counter(s[i:i+p_len]) == p_count:
            ans.append(i)

    return ans
```

时间复杂度：O(n * m)，其中 n 是 s 的长度，m 是 p 的长度。每次 Counter 都要遍历 m 个字符。

#### 优化思路

暴力解法的问题：窗口每移动一步，整个窗口内的字符都要重新统计，实际上只有进出窗口的两个字符发生了变化。

优化方向：
1. **增量更新**：窗口右移时，加入新字符、移除旧字符，只更新这两个字符的计数
2. **用一个变量记录"已满足的字符种类数"**：避免每次都比较整个频次表

#### 最优解法

维护一个定长窗口，用哈希表记录窗口内字符频次，同时用一个计数器 `match` 记录已经完全匹配的字符种类数。当 `match` 等于 p 中不同字符的种类数时，就找到了一个异位词。

时间复杂度：O(n)，空间复杂度：O(1)（字符集大小固定为 26）。

### 4. 关键思维点

1. **异位词的本质**：两个字符串互为异位词，当且仅当它们的字符频次完全相同
2. **定长窗口**：窗口大小始终等于 `len(p)`，每次右移一步，同时加一个字符、减一个字符
3. **match 计数器的妙用**：不用每次比较两个频次表（26 次比较），只需要在字符进出时更新 match 值（O(1) 判断）
4. **边界处理**：如果 `len(s) < len(p)`，直接返回空列表

### 5. 代码实现（Python）

```python
from collections import Counter

def findAnagrams(s: str, p: str) -> list:
    s_len, p_len = len(s), len(p)
    # 特判：s 比 p 短，不可能有异位词
    if s_len < p_len:
        return []

    ans = []
    # 统计 p 中每个字符的频次（需要凑齐的目标）
    p_count = Counter(p)
    # 窗口内字符的频次
    window = Counter()
    # match 记录窗口中已经完全匹配 p_count 的字符种类数
    match = 0
    # p 中不同字符的种类数（match 达到这个值就说明找到了异位词）
    target = len(p_count)

    for right in range(s_len):
        # --- 右边界字符进入窗口 ---
        ch_in = s[right]
        window[ch_in] = window.get(ch_in, 0) + 1
        # 如果这个字符在 p 中，且窗口中的数量刚好等于 p 中的数量
        if ch_in in p_count and window[ch_in] == p_count[ch_in]:
            match += 1  # 又多满足了一种字符

        # --- 左边界字符离开窗口（窗口大小超过 p_len 时触发）---
        if right >= p_len:
            ch_out = s[right - p_len]
            # 如果要移出的字符在 p 中，且移出前刚好匹配
            if ch_out in p_count and window[ch_out] == p_count[ch_out]:
                match -= 1  # 这种字符不再满足了
            window[ch_out] -= 1
            if window[ch_out] == 0:
                del window[ch_out]  # 清理计数为 0 的键

        # --- 判断是否找到异位词 ---
        if match == target:
            # 窗口起始索引 = right - p_len + 1
            ans.append(right - p_len + 1)

    return ans
```

**更简洁的写法**（直接比较 Counter）：

```python
from collections import Counter

def findAnagrams(s: str, p: str) -> list:
    s_len, p_len = len(s), len(p)
    if s_len < p_len:
        return []

    ans = []
    p_count = Counter(p)
    # 先统计第一个窗口
    window = Counter(s[:p_len])

    if window == p_count:
        ans.append(0)

    # 窗口向右滑动
    for i in range(p_len, s_len):
        # 加入右边新字符
        window[s[i]] += 1
        # 移除左边旧字符
        left_char = s[i - p_len]
        window[left_char] -= 1
        if window[left_char] == 0:
            del window[left_char]  # 删除计数为 0 的键，确保比较正确

        # 比较窗口频次和目标频次
        if window == p_count:
            ans.append(i - p_len + 1)

    return ans
```

**定长 vs 变长窗口的区别**：
- **定长窗口**（本题）：窗口大小固定，左右指针同步移动，用 if 判断何时移动左指针
- **变长窗口**（题目3）：窗口大小不固定，用 while 循环收缩左指针直到窗口合法
